Covid

We take Covid seriously. You may be a healthy, fit cyclist, but you can still easily get Covid. It can make you horribly sick and it can kill you, both of which are very bad for your cycling. However, the scientific consensus seems to be that there is very little risk when outdoors, especially when moving around as you are when cycling. So get vaccinated, get outdoors as much as possible, and sign up on the "Contact" page so that we can send you news if things change.
